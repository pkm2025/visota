# 03. Thông tư 200/2014/TT-BTC - Chế độ kế toán DN

> Tùy chọn nâng cao cho DN lớn.

## 1. Tổng quan

| Thông tin | Giá trị |
|----------|--------|
| Số văn bản | 200/2014/TT-BTC |
| Ngày ban hành | 22/12/2014 |
| Ngày hiệu lực | 01/01/2015 |
| Thay thế | QĐ 15/2006/QĐ-BTC, QĐ 48/2006/QĐ-BTC |
| Đối tượng | Mọi DN (lớn, vừa, nhỏ) |
| Bắt buộc | DN lớn; DN nhỏ có thể chọn TT200 hoặc TT133 |

## 2. Điểm khác biệt so với TT133

| Đặc điểm | TT200 | TT133 |
|----------|-------|-------|
| Số lượng TK | ~300 | ~120 |
| Tách TK 641 vs 642 | Có (6412, 6422 chi tiết) | Không tách nhiều |
| Tách TK 511 | 5111, 5112, 5113, 5114, 5117, 5118 | 5111, 5112, 5113, 5117, 5118 |
| Sổ CTGS | Có | Có |
| BCTC | Đầy đủ + chi tiết | Đầy đủ |
| Kế toán HĐTC | 4 chiều (chi tiết) | Đơn giản hơn |
| Tài khoản 138, 338 | Chi tiết hơn | Đơn giản |
| Lợi nhuận | 4211, 4212 (LN chưa PP / LN đã PP) | 421 (gộp) |

## 3. Một số TK khác biệt

| TT200 | TT133 | Mô tả |
|-------|-------|------|
| 641 | – | CP bán hàng (TT133 không có) |
| 6421 | – | CP bán hàng (tách trong TT133 gộp vào 642) |
| 6422 | 642 | CP QLDN |
| 4211 | 421 | LN chưa phân phối |
| 4212 | – | LN đã phân phối (TT133 không có) |
| 5114 | – | Trợ cấp XB (TT133 không có) |
| 2413 | – | Sửa chữa lớn TSCĐ (TT133 không có) |
| 635 | 635 | CP tài chính (TT200 chi tiết: 6351-6358) |

## 4. Hệ thống tài khoản TT200 (subset)

Xem đầy đủ tại: https://docs.kreston.vn/vbpl/ke-toan/che-do-ke-toan/che-do-ke-toan-doanh-nghiep/tt-200-2014-tt-btc/phu-luc-1/

## 5. Sổ kế toán TT200

Số lượng mẫu sổ nhiều hơn TT133:

- **Sổ tổng hợp**: NKC, CTGS, Sổ cái
- **Sổ chi tiết**: Hơn 20 loại
- **Báo cáo tài chính**: B01-DN, B02-DN, B03-DN, B04-DN, B09-DN

## 6. Áp dụng trong phần mềm

### 6.1. Khi nào hỗ trợ TT200?

Khuyến nghị:
- **Phase 1**: Chỉ TT133 (mặc định)
- **Phase 2**: Thêm tùy chọn TT200 cho khách hàng yêu cầu

### 6.2. Cấu trúc dữ liệu

```python
# Setup công ty mới
def setup_company(company, regime='tt133'):
    if regime == 'tt133':
        load_tt133_chart_of_accounts(company)
        load_tt133_report_templates(company)
    elif regime == 'tt200':
        load_tt200_chart_of_accounts(company)
        load_tt200_report_templates(company)
    else:
        raise ValueError(f"Unknown regime: {regime}")
```
